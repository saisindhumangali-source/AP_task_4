# Task 4 - Full Project Implementation

This repository contains three frontend web development projects built using HTML, CSS, and JavaScript.

## Projects

### 1. Personal Portfolio Website
Features:
- Responsive design
- About Me page
- Projects page
- Contact page

### 2. To-Do List App
Features:

- Edit tasks
- Delete tasks
- Mark tasks as completed

### 3. Product Listing Page
Features:

- Search products
- Filter by category
- Sort by price
- Sort by rating
